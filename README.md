# Principios de Algoritmos - UTP 

¡Hola! Este repositorio contiene todos los ejercicios, prácticas y lógica desarrollada durante mi **primer ciclo** en la carrera de Ingeniería en la **Universidad Tecnológica del Perú (UTP)**.

El objetivo de este curso es construir soluciones lógicas a problemas mediante el uso de **pseudocódigos** y **diagramas de flujo**[cite: 1].

## 🛠️ Herramientas Utilizadas
* **PSeInt:** Para la escritura y ejecución de pseudocódigo.
* **Diagramas de Flujo:** Representación gráfica de la lógica algorítmica.
